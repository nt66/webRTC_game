## 安装
```bash
npm install
```
or
```bash
yarn
```


## 运行
```bash
npm start
```

<br/>

## 特性
1）webRTC实现音视频通信 <br />
2) socket.io 处理信令逻辑 <br />
3) pixi执行spine动画

## 本地ip调试浏览器设置
1） 打开 chrome://flags/#unsafely-treat-insecure-origin-as-secure
2） 设置本地ip至框中如：http://xx.xxx.xx.xx:9000
3)  设置属性成为 Enabled
